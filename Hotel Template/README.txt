1. Introduction
---------------
The Hotel Template is built using HTML5, CSS3, and JavaScript.
It is fully responsive and suitable for hotels, booking, and hospitality websites.

2. Installation
---------------
- Unzip the package (if zipped).
- Open the project folder.
- Launch index.html in your browser.

3. File Structure
-----------------
(See Documentation/index.html for the exact project file structure.)

4. Customization
----------------
- Colors: Edit css files in css/css/ and css/media/ as needed.
- Fonts: Update webfonts or add Google Fonts link in the HTML head.
- Logo: Replace img/logo.png
- Images: Replace files in img/ with your own licensed images.
