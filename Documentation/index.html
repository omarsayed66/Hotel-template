Hotel Template - Documentation

Thank you for using the Hotel Template.
This guide will help you to install, use, and customize the template. The file structure below is taken directly from your project folder and matches it exactly.

1. Introduction

The Hotel Template is built using HTML5, CSS3, and JavaScript. It is fully responsive and designed for hotels, booking websites, and hospitality services.

2. Installation

  1. Unzip the package (if zipped).
  2. Open the project folder and launch index.html in your browser (located in the root of the project).

4. Customization

  • Colors: Edit CSS files inside css/css/ or main-style files inside css/css/main-style.css.
  • Fonts: Fonts are available inside webfonts/ and webfonts-css. You can change them or link external Google Fonts.
  • Logo: Replace the logo image file inside the img/ folder.
  • Scripts: Custom JS files are inside the js/ folder (e.g., main.js, slide.js, etc.).
